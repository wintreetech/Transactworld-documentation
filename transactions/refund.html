Refund API

  • Endpoint: POST
  • Content Type : application/json
  • Description : This API is used to process refunds for a given transaction. Merchants can initiate partial or full refunds by providing the transaction ID (TSID), amount, and refund type.

Request Parameters

Parameter Type Required Description Example
merchantId Number Yes Unique ID of the merchant initiating the refund 6
amount Number Yes Amount to be refunded 10
tsId Number Yes Transaction Session ID of the original payment 423623
refundType String Yes Type of refund (Manual / Auto) Manual

Request Body


										


										

											
curl --location 'https://example.com/transactionUtilService/transactionDetails/refund' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <your-token>' \
--data '{
    "transactionActionVO": {
        "merchantId": 6,
        "amount": 10,
        "tsId": 423623,
        "refundType": "Manual"
    }
}'
