Introduction

  • Endpoint: POST
  • Content Type : application/json
  • Description : Fetches the transaction details for a given Transaction Session ID (TSID). This API is used to retrieve information related to a specific transaction, including status, amount, currency, payment method, timestamps, and other metadata. This is useful for merchants, support teams, or backend systems that need to view transaction-related data for reconciliation, status checking, or reporting.

Request Parameters

Parameter Type Required Description Example
