Introduction

  • Endpoint: POST
  • Content Type : application/json
  • Description : Initiates a payment transaction with comprehensive customer details, shipping information, and payment method configuration. Supports multiple payment types including credit cards, digital wallets, and UPI transactions.

Parameter Type Required Description Example
memberId string Yes Unique identifier for the merchant member Accept only numeric value (e.g., 12345)
